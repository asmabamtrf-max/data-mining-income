# Adult Income Data Mining Project

## Overview
A data mining project using the Adult dataset to analyze and predict income levels.

## Techniques
- Data preprocessing (handling missing values, normalization, discretization)  
- Classification (J48 Decision Tree, Naive Bayes)  
- Association rules (Apriori algorithm)  

## Tools
- WEKA  

## Key Work
- Cleaned and prepared dataset  
- Applied classification models  
- Evaluated performance using accuracy, precision, recall, and F1-score
